ZDialog

ZVC presents Modal Dialogs independent of the CSS framework used. Some specific code needs to be included in your index.html to get things working.

If your Boostrap application uses Modal Dialogs, after zvc.js import, include in your index.html
<script src="zvc/zvc-bootstrap-dialogs.js"></script>

Each Dialog should be designed as a Component, with its View (HTML) and Controller (JS). HTML can reference another ZVC components in the usual way (Statically o Dynamically). Controller shouls extend ZDialog instead ZCustomController.

Dialogs should be closed from Dialog Controller using this.close(returnData) or this.cancel()

Use this.showDialog(path, options, okCallback(returnData), cancelCallback()) to open a Dialog. Options will be passed to onThis_init(options) in Dialog controller.
