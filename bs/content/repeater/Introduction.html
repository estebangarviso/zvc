ZVC - Bootstrap Paginated Repeater

ZVC Bootstrap Paginated Repeater extends ZRepeater component adding pagination behaviour. In order to use this component, please include
<script src="zvc/zvc-bootstrap-repeater.js"></script> after zvc.js

Examples in next sections show how pagination works. Examples use a simulated Customers Model to interact with data. This model supports search and basic CRUD operations. A text search field is also used in examples.

Customers are rendered in page using CustomerCard component. One instance of this component is created for every row in the page. Field "repeaterIndex" starts with 0 for each returned page. Field "controllers" in repeater components hols and array of controller instances created for current page.
