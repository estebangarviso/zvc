ZVC Bootstrap Paginated Repeater - Server Side Pagination

Server pagination should be used for large number of records.

Server Side Pagination is based on two events triggered from repeater, one to obtain the total numbers of rows (one call for each call to refresh) and another to get rows for each displayed page.

Services implementing pagination could use databses offset / limit combination or equivalents techniques to retrieve ranges of rows.

To enable server side pagination, use data-z-pagination="server" in pagination component.
