ZVC Bootstrap Table - Local Pagination

Local Pagination retrieves all rows from server and pagination is handled in the browser. It can reduce performance in initial loading but accelerate navigation between pages.

Pagination in ZVC Bootstrap Tables requires definition of a "pagination" bootstrap component and, optionally, a display area to show visible rows resume to the user.

As data-z attributes in pagination component, number of rows per page and number of pages in paginator can be configured.

Data retrieval in local pagination works the same as no pagination, using getRows event after calling refresh method in table.
