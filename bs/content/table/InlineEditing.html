ZVC Bootstrap Table - Inline Record Edition

ZVC Bootstrap Table has the ability to open a "details" area for each displayed row, and even for a new row to be added. This area can be filled with any Custom component in ZVC.

Row detail area can be used to show row properties, offer custom actions based on row data or, as in this example, to edit the row data.

One of the columns defined in THEADER can act as an "opener" for row's details area. When user selects the icon in that column, table asks to container (using an event) for the path to the component to open and de initial options to pass to that component after creation (on its init event).

This demo uses a Custom Component called PEdCustomer (Panel for Customer Edition) equivalent (and ver similar) to WEdComponent used in Dialog Based Edition. This panel has three buttons: Save, Cancel and Delete. Delete Button is hidden when panel is open to edit a new record. Same way, Customer Id field is disabled when editing an existing record.

Events triggered from Details Component are passed directly to table container controller, adding to extra parameters at the begining of the handler function: row and rowIndex.
