ZVC Bootstrap Table - Custom Clickable Columns

Cells are rendered with the value of each row. If corresponding field contains HTML instead of plain text, it will be rendered as HTML.

If attribute data-z-clickable is added to required TH elements, table will trigger a cellClick(row, rowIndex, field) event.
Cell click events, inconjuntion with custom HTML cells, can be used to change some data in the table.

Following example uses this combination to change the status of each customer (active / inactive) just by clicking an icon in the last column.
A common pattern is to use a prepareRow method to add custom HTML to a row returned by a service.

If the row passed to a table contains a field called _rowClass the entire row (TR element) will have that style. It can be used to show a record state. In this example, inactive customers will show with a different style.

Finally, if you want to update a row in a table without refreshing all rows (no need to call server again) you can use updateRow(rowIndex, row)
