ZVC - Bootstrap Table

ZVC Bootstrap Table is based on standard HTML table plus pagination component from Bootstrap. In order to use this component, please include
<script src="zvc/zvc-bootstrap-table.js"></script> after zvc.js

Tables can be used with or without pagination. Pagination can be local or sever side. It also supports a custom component to use as editor, details viewer, etc. for each row in table.

Please follow examples in next sections in order to see all funcionalities provided by ZVC Bootstrap Table. Examples use a simulated Customers Model to interact with data. This model supports search and basic CRUD operations.
