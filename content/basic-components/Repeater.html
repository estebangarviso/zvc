ZRepeater

Repeater elements can be DIV, UL, OL. For each row received, an instance of a referenced controller (and its view) is created. Row object is passed to each child instance as the argument in it's "init" event.

Use "data-z-repeat" attribute to reference target element. Each instance of target element will be contained inside a DIV element. Attribute "data-z-container-tag" can be used to change container element type. "data-z-container-classes" attribute can be used to define CSS classes for each container instance.

Created controllers for each row have an attribute called "repeaterIndex". Child controller instances can be accesed using "controllers" array field in repeater control.
