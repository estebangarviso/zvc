ZInput

ZInput controllers are created for every INPUT and TEXTAREA element in HTML with a valid "id" attribute.

value and checked properties are added (to base ZController) to access the state of the input or text area field.

Basic Inputs Demo
Change Event triggered as user types

ZInput can be parametrized to detect change events without loosing its focus. Use data-z-autochange-delay="300" to trigger a change event if the user does not type anything en 300 ms.
