ZVC - Anatomy of index.html

ZVC is "style agnostic", so you can use your favourite CSS framework to design your application.

The only required file to use the framework is
<script src="zvc.js"></script>

Some components are provided for specific frameworks that offers graphics parts with its own styles. For example, a Bootstrap table with paginator and a section describing current page and rows range is provided as an external (optional) component that can be included in your project using:
<script src="zvc-bootstrap-table.js"></script>

After the index.html page has finished loading its content, a small script is needed to create the root component (usually one) representing the Application. Event document.DOMContentLoaded can be used to this purpose. Root component must be associated to a DOM element in index.html page. 


    <script>
        document.addEventListener("DOMContentLoaded", _ => {                
            ZVC.options.debug.events = false;
            ZVC.fromElement("main");
        });
    </script>
